# sdd-central-skills
Repositório para conter as skills padronizadas para reaproveitamento nos demais projetos

## Skills disponíveis

- `architecture/hexagonal`: arquitetura hexagonal para fluxos de Spec Driven Development com Speckit.
- `testing/junit5`: criação e revisão de testes Java com JUnit 5 a partir de specs e critérios de aceite.
- `testing/mockito`: uso de Mockito para isolar portas, dependências e colaboradores em testes Java.
- `java-21`: idiomatismos e decisões de implementação para Java 21.
